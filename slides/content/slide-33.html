4. Quản lý Tác vụ - asyncio.Task

async def background_job():
    try:
        print("-> [Task]: Đang chạy ngầm...")
        await asyncio.sleep(5)
        return "Xong!"
    except asyncio.CancelledError:
        print("-> [Task]: Đã bắt được sự kiện HỦY!")
        raise
async def main():
    task = asyncio.create_task(background_job())
    await asyncio.sleep(1)
    
    print(f"Hoàn thành chưa? {task.done()} | Yêu cầu hủy...")
    task.cancel()  # Kích hoạt CancelledError
    
    try: await task
    except asyncio.CancelledError:
        print("Đã xác nhận hủy thành công.")

Định nghĩa Task & So sánh

asyncio.Task: Con của asyncio.Future, dùng để đóng gói và quản lý trạng thái của coroutine.

So sánh: Coroutine là bản thiết kế tĩnh. Task = Coroutine + Đã lên lịch + Theo dõi trạng thái.
