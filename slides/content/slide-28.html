Phần 03 · asyncio

Lấy và điều khiển Event Loop

import asyncio

async def greet():
    await asyncio.sleep(0.5)
    print("Xin chào từ Coroutine!")

# Lấy (hoặc tạo) một Event Loop
loop = asyncio.get_event_loop() 

# Đặt coroutine vào queue và chạy
loop.create_task(greet()) # Đăng ký task
loop.run_until_complete(asyncio.sleep(1))
loop.close()

Kết quả đầu ra Terminal (Output)

Xin chào từ Coroutine!

Các phương thức điều khiển quan trọng

  • create_task(coro): Đăng ký một coroutine vào Event Loop và bắt đầu lên lịch chạy ngay lập tức dưới dạng Task.
  • call_soon(callback, *args): Lên lịch gọi hàm callback thông thường càng sớm càng tốt khi loop rảnh.
  • call_later(delay, callback, *args): Lên lịch gọi hàm callback sau một khoảng thời gian delay giây.
  • stop(): Dừng vòng lặp sự kiện đang hoạt động.
  • is_running(): Kiểm tra Event Loop có đang chạy hay không.
