Mô phỏng: executor.submit()

Làm sao để chúng ta gửi tác vụ vào ThreadPool và lấy kết quả mà không bị chặn chương trình chính? Đáp án chính là Future.

Mô phỏng Code Từng Dòng

with ThreadPoolExecutor(max_workers=2) as executor:
    print("Gửi tác vụ...")
    future1 = executor.submit(task, "A")
    future2 = executor.submit(task, "B")
    
    print("Main thread vẫn chạy!")
    
    # Đợi và lấy kết quả
    print(future1.result())
    print(future2.result())
