1. Coroutine (Hàm Bất Đồng Bộ)

Sự khác biệt cốt lõi giữa def và async def

Hàm Thường (def)

Gọi là chạy ngay lập tức

def nau_an():
    return "Món ăn"

# Có đồ ăn ngay!
ket_qua = nau_an()

Trả về: Dữ liệu thực tế

Hàm Async (async def)

Gọi KHÔNG chạy ngay, trả về "Công thức"

async def nau_an():
    return "Món ăn"

# Chưa có đồ ăn đâu!
ket_qua = nau_an()

Trả về: Coroutine Object

(Một bản công thức, cần Event Loop để nấu)

Chốt lại: Viết async def là bạn đang tạo ra một Coroutine (Tác vụ chờ). Phải đưa nó cho Event Loop thì nó mới chạy!
