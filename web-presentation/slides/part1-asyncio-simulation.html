2.7 Mô phỏng luồng thực thi

Bản chất của await là đóng băng bộ nhớ cục bộ, giải phóng Event Loop theo dõi I/O.

1. Mã lệnh đang chạy

async def task_A():
print("A bắt đầu")
await asyncio.sleep(2)
print("A hoàn thành")

async def task_B():
print("B bắt đầu")
await asyncio.sleep(2)
print("B hoàn thành")

async def main():
# Chạy đồng thời cả 2 task
await asyncio.gather(
task_A(),
task_B()
)

# Điểm khởi động toàn bộ chương trình
asyncio.run(main())
