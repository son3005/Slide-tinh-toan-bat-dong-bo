Phần 3: Concurrent (Đa luồng/Đa tiến trình)

3.1 Thư viện concurrent.futures

Đây là một module chuẩn trong Python cung cấp giao diện cấp cao (high-level API) để khởi chạy các tác vụ bất đồng bộ.

Hai công cụ cốt lõi (Executors):

  • ThreadPoolExecutor: Quản lý một nhóm các Threads (luồng). Tốt cho các tác vụ I/O-bound (mạng, đọc file).
  • ProcessPoolExecutor: Quản lý một nhóm các Processes (tiến trình độc lập). Bắt buộc phải dùng cho CPU-bound (tính toán nặng).

Tại sao lại phân tách rõ ràng như vậy? Điều này liên quan mật thiết đến một cơ chế bảo vệ cực kỳ khét tiếng của Python: GIL (Global Interpreter Lock).

Kho lưu trữ Code (Rehearsal Store)

Mẫu code cơ bản để khởi tạo một ThreadPool. Hãy lưu lại để ôn tập nhé!

from concurrent.futures import ThreadPoolExecutor
import requests

def fetch_url(url):
    return requests.get(url).text

with ThreadPoolExecutor(max_workers=5) as executor:
    # Submit task to the pool
    future = executor.submit(fetch_url, "http://example.com")
    print(future.result())
