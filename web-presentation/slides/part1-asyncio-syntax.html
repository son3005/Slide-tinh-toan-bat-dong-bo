Phần 2: Asyncio (Đơn luồng)

2.4 Cú pháp cơ bản & Cơ chế hoạt động

Cú pháp async / await giúp viết mã bất đồng bộ gần giống như mã tuần tự thông thường, rất dễ đọc và dễ bảo trì.

Sự kì diệu của từ khóa await

Khi gặp lệnh await, luồng xử lý không bị đóng băng. Thay vào đó, nó thực hiện 3 bước:

1. Tạm dừng (Yield) Tự nguyện từ bỏ quyền kiểm soát CPU, trả lại cho Event Loop.
2. Lưu trữ (Save State) "Đóng băng" toàn bộ biến cục bộ và dòng code hiện tại vào RAM.
3. Khôi phục (Resume) Khi I/O xong, Event Loop khôi phục State và chạy tiếp.

Python: Chạy đồng thời (Concurrent)

import asyncio

async def fetch_data(id):
    print(f"Đang tải {id}...")
    await asyncio.sleep(2) # Giải phóng CPU
    return f"Dữ liệu {id}"

async def main():
    # Trường hợp đặc biệt: Gom nhóm chạy TẤT CẢ cùng lúc!
    # Thay vì mất 6s, ta chỉ mất 2s để tải xong cả 3 file.
    results = await asyncio.gather(
        fetch_data(1),
        fetch_data(2),
        fetch_data(3)
    )
    print(results)
