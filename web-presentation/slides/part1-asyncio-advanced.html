2.5 Bách khoa Cú pháp (API)

Bên cạnh await, Asyncio cung cấp một kho vũ khí (API) mạnh mẽ để bạn thao túng các Task theo mọi kịch bản phức tạp nhất.

Điểm kích nổ Event Loop

Bạn không thể gọi trực tiếp một hàm async bằng cú pháp thông thường. Hàm run() là cầu nối giữa thế giới đồng bộ (Sync) và bất đồng bộ (Async).

import asyncio

async def main():
    print("Hello from Asyncio!")

# Sai: main() sẽ không chạy! Nó chỉ trả về coroutine.
# main() 

# Đúng: Khởi tạo Event Loop, nhét main() vào và chạy
asyncio.run(main())

Chạy ngầm (Background Tasks)

Giúp bạn kích hoạt một tác vụ chạy ngầm trong background mà không cần phải đứng lại đợi nó xong (Fire and Forget).

async def gui_email():
    await asyncio.sleep(5)
    print("Đã gửi email!")

async def main():
    # Đẩy việc gửi email ra chạy ngầm
    task = asyncio.create_task(gui_email())
    
    # Hàm main vẫn tiếp tục chạy lập tức
    print("Phản hồi cho user xong!")
    
    # (Sau này có thể await task nếu muốn lấy kết quả)

Gom nhóm và Đợi (Concurrency)

Đẩy cùng lúc N tác vụ vào Event Loop và đợi tất cả chúng hoàn thành. Đây là trái tim của việc tối ưu thời gian.

async def tai_file(id):
    await asyncio.sleep(2)
    return f"Data {id}"

async def main():
    # Cả 3 file sẽ được tải cùng lúc (song song)
    # Tổng thời gian: 2 giây (thay vì 6 giây)
    ket_qua = await asyncio.gather(
        tai_file(1),
        tai_file(2),
        tai_file(3)
    )
    # ket_qua = ['Data 1', 'Data 2', 'Data 3']

Kiểm soát Timeout

Không ai muốn chờ đợi mãi mãi. Ép một tác vụ phải hoàn thành trong N giây, nếu không sẽ tự động hủy (Cancel) nó.

async def goi_api_cham():
    await asyncio.sleep(10) # API bị kẹt

async def main():
    try:
        # Chỉ cho phép API chạy tối đa 3 giây
        await asyncio.wait_for(goi_api_cham(), timeout=3.0)
    except asyncio.TimeoutError:
        print("Quá lâu! Đã tự động hủy tác vụ.")

Xử lý ngay khi có kết quả

Khác với gather() phải đợi TẤT CẢ xong mới trả về cục bộ. as_completed() cho phép Task nào xong trước thì đem ra dùng luôn.

async def tai_file(time, ten):
    await asyncio.sleep(time)
    return ten

async def main():
    tasks = [tai_file(5, "Nặng"), tai_file(1, "Nhẹ")]
    
    # Vòng lặp này sẽ in "Nhẹ" ra trước (sau 1s),
    # rồi mới in "Nặng" ra sau (sau 5s).
    for coroutine in asyncio.as_completed(tasks):
        ket_qua = await coroutine
        print(ket_qua)
