4.1 Khi Asyncio gặp Blocking

Nút Thắt Cổ Chai (Bottleneck)

Bạn đang có một hệ thống Asyncio cực nhanh có thể xử lý 10,000 kết nối. Nhưng đột nhiên bạn cần dùng thư viện requests (hoặc tính toán nặng) bên trong một Coroutine.

Khổ nỗi, requests.get() là hàm Đồng bộ (Synchronous/Blocking). Nó không biết "nhường CPU". Nó bắt toàn bộ chương trình phải đứng im chờ đợi!

Hậu quả:

Event Loop bị treo cứng. 9,999 kết nối khác bị "đứng hình" theo chỉ vì một tác vụ tải web/ảnh!

Đoạn Code "Tự hủy"

Không bao giờ được gọi hàm Blocking trực tiếp bên trong `async def`.

import asyncio
import requests

async def download_image():
    # LỖI: Lệnh này CHẶN cứng Event Loop!
    # Nó không có chữ `await`, nó không nhường CPU.
    response = requests.get("https://...")
    return response.content

async def main():
    # Cả hệ thống 10,000 users đứng im chờ...
    await download_image()
