Phase 3: Khi Asyncio gặp Blocking

Vấn đề chết người khi gọi hàm đồng bộ (như thư viện requests) bên trong Event Loop

Nút Thắt Cổ Chai

Bạn đang có một hệ thống Asyncio cực nhanh có thể xử lý 10,000 kết nối. Nhưng đột nhiên bạn cần dùng thư viện requests để tải một tấm ảnh.

Khổ nỗi, requests.get() là hàm Đồng bộ (Synchronous/Blocking). Nó bắt toàn bộ chương trình phải đứng im chờ đợi!

Hậu quả:

Event Loop bị treo cứng. 9,999 kết nối khác bị "đứng hình" theo chỉ vì một tác vụ tải ảnh!

❌ Đoạn code tự hủy
import asyncio
import requests

async def download_image():
    # LỖI: Lệnh này CHẶN cứng Event Loop!
    response = requests.get("https://...")
    return response.content

async def main():
    # Cả hệ thống đứng im chờ 1 file...
    await download_image()
